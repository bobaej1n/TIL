# floor
### 1. 요약
소수점을 가장 가까운 정수로 내림한 값을 리턴
<br /><br />

### 2. 문법
```javascript
Math.floor(x)
```
<br />

### 3. 인자
|인자명|데이터형|필수/옵션|설명|
|---|---|---|---|
|x|number|필수|내림할 수|
<br />

### 4. 반환값
number, 정수
<br /><br />

### 5. 예제
```javascript
alert(Math.floor(5.1)); // number, 5
alert(Math.floor(5.9)); // number, 5
alert(Math.floor(5)); // number, 5
alert(Math.floor(-5.1)); // number, -6
alert(Math.floor(-5.9)); // number, -6
```
